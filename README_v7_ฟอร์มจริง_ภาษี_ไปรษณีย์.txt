ThanyawitCustomer iPad Xcode v7

เพิ่มใน v7:
1) PDF แบบฟอร์มจริงใกล้เคียงตัวอย่าง
   - ใบแจ้งหนี้
   - ใบเสร็จรับเงิน / ใบกำกับภาษี
   - ใบส่งมอบงานจ้าง
   ใช้วิธีสร้าง layout ใหม่ แล้วพิมพ์แทรกข้อความลงตำแหน่ง

2) หัก ณ ที่จ่าย / Timeline ภาษี
   - เก็บฐานยอดก่อน VAT
   - เก็บยอดหัก ณ ที่จ่าย
   - เก็บสถานะภาษีและหลักฐาน เพื่อใช้ขอคืนภาษี

3) Document Control / รับส่งไปรษณีย์
   - สร้างสำเนารายการเอกสาร
   - กรอกเลขพัสดุ
   - ตั้งวันตรวจสถานะหลัง 2 วันทำการ
   - บันทึกสถานะจัดส่งไว้หลังบ้าน

4) เลขเอกสาร
   - เริ่ม run ต่อจากตัวอย่างเลข 015 เป็น 016
   - เก็บ sequence ในเครื่อง iPad

วิธีใช้:
- เปิดเมนู ส่งออก > Export ฟอร์มเอกสารจริง PDF
- ถ้าต้องการบันทึกหลังบ้าน ให้กด "บันทึกสำเนาเข้า Document Control"
- เมนู Timeline ภาษี ใช้แก้สถานะภาษี/หลักฐาน
- เมนู รับส่งไปรษณีย์ ใช้กรอกเลขพัสดุและสถานะจัดส่ง

หมายเหตุ:
- ส่วนสีฟ้าในฟอร์มทำแบบใกล้เคียง ไม่ใช่ clone ภาพสแกน 100%
- การดึงสถานะไปรษณีย์อัตโนมัติจริงต้องต่อ API ของผู้ให้บริการขนส่งใน v8
