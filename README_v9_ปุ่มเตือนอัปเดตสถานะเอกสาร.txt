ThanyawitCustomer iPad Xcode v9

เพิ่มตามคำสั่ง:
- ทำหมายเหตุ/แจ้งเตือนเป็นปุ่มกด
- หน้า Document Control มีแถบเตือน "ต้องกดอัปเดตสถานะเอกสาร"
- มีตัวเลขจำนวนเอกสารที่ต้องกดอัปเดต
- มีปุ่ม "กดเพื่ออัปเดตสถานะทั้งหมดที่ต้องเช็ก"
- มี Toggle "โชว์เฉพาะที่ต้องกด"
- แต่ละรายการที่ต้องเช็กมี badge "ต้องกด"
- หน้า Dashboard โชว์เตือนว่ามีเอกสารต้องอัปเดตสถานะ

วิธีใช้:
1) เข้าเมนู รับส่งไปรษณีย์
2) ถ้ามีแถบสีส้ม แปลว่าต้องกดอัปเดต
3) กด "กดเพื่ออัปเดตสถานะทั้งหมดที่ต้องเช็ก"
4) หรือเปิดรายการทีละอัน แล้วกด "ต้องกดอัปเดต"
5) ระบบบันทึกวันที่ตรวจและสถานะไว้หลังบ้าน

หมายเหตุ:
- ถ้าไม่มี token/API จะเตือนให้กรอก token หรือใช้ปุ่มเปิดเว็บติดตาม
- ไปรษณีย์ไทยใช้ token ในเมนูตั้งค่าขนส่ง
- Flash Express ถ้ายังไม่มี endpoint ให้ใช้เปิดเว็บติดตามก่อน
