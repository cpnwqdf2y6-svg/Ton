ThanyawitCustomer iPad Xcode v6 PDF Export

เพิ่มใน v6:
- Export Billing PDF
- Export Customers PDF
- PDF ใส่โลโก้บริษัท, ชื่อบริษัท, วันที่เอกสาร, สถานะพร้อมออก/รอตรวจ
- CSV ยังอยู่เหมือนเดิม

วิธีใช้:
1) ปิด Xcode เดิม
2) แตก ZIP v6
3) เปิด ThanyawitCustomer.xcodeproj
4) Signing & Capabilities เลือก Team เดิม
5) Product > Clean Build Folder
6) กด Run
7) เข้าเมนู "ส่งออก" ในแอป แล้วกด Export Billing PDF หรือ Export Customers PDF

หมายเหตุ:
- PDF นี้เป็นรายงานพิมพ์ออกอ่านง่าย
- ยังไม่ใช่ใบแจ้งหนี้/ใบส่งมอบงานตามฟอร์มบริษัท 100%
- ถ้าต้องการฟอร์มบริษัทจริง ให้ทำต่อเป็น v7 ฟอร์ม PDF เอกสารจริง
